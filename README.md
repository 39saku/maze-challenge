# Maze Challenge

このプロジェクトは、中高生向けのプログラミング体験ワークショップ用カリキュラムです。
JavaScriptを使って、キャラクターをゴールへ導くアルゴリズムを作成し、最短手数でのクリアを目指します。

## 📁 フォルダ構成
- `index.html`: 実行用メインファイル。ブラウザで開いて使用します。
- `student/`: **★皆さんが編集する場所です★**
  - `student_code_1.js`: 問題1の解答用
  - `student_code_2.js`: 問題2の解答用
  - `student_code_3.js`: 問題3の解答用
- `core/`: 迷路の動きを制御する大事なプログラムが入っています（編集禁止）。
- `data/`: 迷路の形（マップ）のデータが入っています。

## 🚀 使い方
1. このフォルダをダウンロードまたはクローンします。
2. `index.html` をブラウザ（Google Chrome, Arc, Safariなど）で開きます。
3. 自分のニックネームを入力します。
4. 解きたい問題のボタン（問題1〜3）を押し、「実行」ボタンでプログラムを走らせます。

## 💻 プログラミング・リファレンス
`student/` フォルダ内の各ファイルにプログラムを記述します。

- `await move("方向")`: 1マス移動します。
  - 方向は `"up"`, `"down"`, `"left"`, `"right"` のいずれかです。
- `await dash("方向")`: 指定した距離（運営が設定）を一気に進みます。
  - 途中で壁にぶつかるとペナルティが発生します。
- `look("方向")`: 指定した方向に何があるかを確認します。
  - `1` = 壁, `0` = 通路, `2` = コイン, `3` = 落とし穴, `4` = ゴール
- `await sleep(ミリ秒)`: 指定した時間だけ動きを止めます。

## 🏆 ルール
1. **最短手数**でゴールを目指してください。
2. 迷路内の**すべてのコイン**を回収しないとゴールできません。
3. **落とし穴**を踏むと数秒間動けなくなります。
4. 何度でも書き直して「リセット」→「実行」が可能です。
